Minimum Requirements

  • OS: Windows 7/8/10/11
  • Processor: 2 GHz Dual Core
  • Memory: 2 GB RAM
  • Graphics: 512 MB DirectX 9.0c compatible
  • Storage: 2 GB available space
  • The Guild 2: Renaissance (Steam version)

Recommended Requirements

  • OS: Windows 10/11
  • Processor: 3 GHz Quad Core
  • Memory: 4 GB RAM
  • Graphics: 1 GB DirectX 11 compatible
  • Storage: 3 GB available space
  • The Guild 2: Renaissance (Steam version)

Installation Instructions

  1. Ensure you have The Guild 2: Renaissance installed via Steam
  2. Download the Reforged Mod installer from the links below
  3. Close the game if it's running
  4. Run the installer and follow the on-screen instructions
  5. Choose whether to back up your original game files (recommended)
  6. Allow the installer to complete the installation process
  7. Launch the game through Steam as normal

Important Notes

  • The Reforged Mod is compatible with save games from the vanilla game, but for the best experience, we recommend starting a new game.
  • If you encounter any issues, please check our Troubleshooting Guide or visit our Discord for support.
  • The installer includes an uninstall option if you wish to revert to the vanilla game.
